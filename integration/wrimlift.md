# WRASCI — Intégration WRIMLIFT

## Dans une entrée lexicale

```wriml
^le id="gba_01":
    ^lx:
        ^fm lng="ega-Latn": gbà _fm:
    _lx:
    ^ph:
        ^wa: gb a tl _wa:
    _ph:
    ^sn id="gba_01_1":
        ^ps value="v"*
        ^ge lang="fr": boire _ge:
    _sn:
_le:
```

## Comparaison avec ^chr*

`^chr*` est utilisé pour insérer des caractères unicode ponctuels.
`^wa:..._wa:` est utilisé pour des séquences phonétiques complètes.
Les deux coexistent sans conflit.
